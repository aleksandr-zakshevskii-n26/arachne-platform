## Opportunity: Visual Fraction Arithmetic Tool

**Target user:** Children aged 8–12 struggling with fraction addition and subtraction, and their parents helping with homework.

**Problem:** Fraction arithmetic is the #1 stumbling block in elementary math. Kids can't see *why* 1/3 + 1/4 = 7/12 — the LCD step feels like arbitrary magic. Parents search for fraction help constantly but existing tools either show no visuals (Mathway, Symbolab), show static diagrams with no worked steps (Math is Fun), or require complex interaction (Visnos). No web tool lets a child type a fraction problem and immediately see proportional colored bars walking through each step: original fractions → equivalent fractions with common denominator → combined result.

**Our angle:** Type two fractions and an operation. Get a step-by-step visual explanation using proportional colored bars — bars subdivide to show equivalent fractions, then slide together to show the result. One concept, one screen, instant "oh, I see why the denominator changes." The visual fills a gap no current tool covers: animated or static bars tied directly to arithmetic steps.

**MVP scope:** Single HTML file, no backend. Three input fields: first fraction (e.g. "1/3"), operator (+, −, ×, ÷ selector), second fraction (e.g. "1/4"). Output: a vertical sequence of labeled steps, each with proportional SVG colored bars and the corresponding equation. Step 1: original fractions drawn as bars. Step 2: equivalent fractions with LCD — bars subdivided into smaller segments. Step 3: result — segments combined/highlighted with the answer. **Estimated build time ≤ 10 minutes** for the static bar version (proportional SVG rectangles, no animation). Animation (bars splitting, sliding) is a v1.1 enhancement. Core logic is LCD calculation, equivalent fraction conversion, and drawing proportional `<rect>` elements — all straightforward.

**Why now:** Fraction help is an evergreen, high-volume search term. Homeschool populations have grown 50%+ since 2020 and these parents actively seek single-purpose tools. No competitor has combined worked-step arithmetic with proportional bar visuals in a clean, free web tool. The gap is open.

**Risks:** (1) "Fraction calculator" is a crowded keyword — SEO will be hard without the visual hook being immediately visible in sharing/screenshots. (2) Static bars without animation may underwhelm — the "aha" is stronger with motion. (3) Kids may use it as an answer machine rather than learning tool; the step-by-step design mitigates this but doesn't eliminate it. (4) Multiplication and division of fractions are harder to visualize with bars — MVP should focus on addition/subtraction only.

**Recommended next step:** Product director to make a go/no-go call on building a static-bar MVP (addition and subtraction only, single HTML file). If go, assign to developer with explicit instruction: static proportional SVG bars, no animation in v1, three inputs, four visual steps. Ship in one session.
