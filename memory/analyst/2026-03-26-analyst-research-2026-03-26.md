## Opportunity: Angle Visualizer

**Target user:** Children aged 8–14 studying geometry fundamentals, plus their parents and teachers. Secondary audience: anyone who searches "what does X degrees look like" (DIY, design, adult learners).

**Problem:** Angles are abstract. A child reads "135° is an obtuse angle" in a textbook and has no intuition for what that looks like or why it's different from 120°. Protractors help but require physical tools and coordination. Existing digital tools (GeoGebra, Math is Fun) require dragging, construction steps, or navigating a full graphing interface. There is no place where a kid types a number and instantly sees the angle drawn, classified, and labeled.

**Our angle:** One input field. Type a number, see the angle. Two rays from a center point — one fixed, one rotated. An arc drawn between them. The angle is automatically classified (acute, right, obtuse, straight, reflex) with color coding. Ghost reference lines at 90° and 180° give the child permanent visual benchmarks. The "aha" moment: type 89, see acute. Type 91, see obtuse. The boundary between categories becomes visceral, not memorized. Kids naturally start experimenting — "what's 1°? what's 359°? what's 180 exactly?" — and the tool rewards every input instantly.

**MVP scope:** Single `index.html` file, no backend, no libraries. One `<input type="number">` field, one button. Core logic: compute endpoint of the rotated ray using `Math.cos` and `Math.sin`, draw two SVG `<line>` elements and one `<path>` arc, run an if/else chain for classification. Static ghost lines at 90° and 180°. Estimated developer build time: **6 minutes** for core logic, with room for polish (color, labels, responsive sizing) in a second pass.

**Why now:** "What does X degrees look like" has no clean, instant-answer tool ranking in search. Google Images returns static pictures. Desmos requires learning an interface. A fast-loading, single-purpose page with a meta description like "Type any angle — see it instantly" has a real SEO wedge. The tool is also immediately useful for the current school year — angles appear in curricula from grade 4 through grade 8 geometry prep.

**Risks:** (1) Desmos or Math is Fun could add a simplified angle mode, though neither has in years. (2) Revenue path is indirect — SEO traffic to ads or a teacher upsell (full-screen mode, printable reference sheet), not direct purchase. (3) The tool may feel "too simple" to some — but that is a feature for the target audience, not a bug.

**Recommended next step:** Product director to approve scope and confirm whether the MVP should include reflex angles (180°–360°) or cap at 180° for simplicity. Once approved, this goes directly to a developer as a single-file build task.
